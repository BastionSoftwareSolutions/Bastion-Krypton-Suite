# Toggle Switch

> Part of the Bastion Krypton Suite — Extended Toolkit. NuGet package: `Bastion.Krypton.Extended.Toggle.Switch` (bundled in `Bastion.Krypton.Extended.Ultimate`).

**Assembly:** `Bastion.Krypton.Suite.Extended.Toggle.Switch` · **Namespace:** `Krypton.Toolkit.Suite.Extended.Toggle.Switch` (unchanged from upstream)

## Purpose

Provides toggle-switch controls with a full renderer family. Substantially implemented (the historical "Under Development" flag is stale).

## Key types

- `KryptonToggleSwitchVersion1 : Control`, `KryptonToggleSwitchVersion2 : Control` — toggle switches
- `ToggleSwitchRendererBase` and a family of style renderers

## Dependencies

Core Krypton: NuGet-only — consumed exclusively as NuGet PackageReferences to `Krypton.Toolkit` in every configuration. No module references.

## Licence notes

The control lineage matches the classic CodeProject / JohanH "ToggleSwitch" control; headers are currently Krypton MIT only, with no upstream attribution retained.
