# Navigator (Extended)

> Part of the Bastion Krypton Suite — Extended Toolkit. NuGet package: `Bastion.Krypton.Extended.Navigator` (bundled in `Bastion.Krypton.Extended.Ultimate`).

**Assembly:** `Bastion.Krypton.Suite.Extended.Navigator` · **Namespace:** `Krypton.Toolkit.Suite.Extended.Navigator` (unchanged from upstream)

## Purpose

Extends the Krypton Navigator with editor and Outlook-bar style navigation controls.

## Key types

- `KryptonNavigatorButton : KryptonButton` — navigator button
- `KryptonNavigatorEditor : KryptonNavigator` — navigator editor
- `OutlookBar : Control`, `OutlookBarNavigationPaneOptions : KryptonForm` — Outlook-bar navigation
- `CustomButton : Button` — custom button

## Dependencies

Core Krypton: NuGet-only — consumed exclusively as NuGet PackageReferences to `Krypton.Toolkit` and `Krypton.Navigator` in every configuration. Module references: Drawing.
