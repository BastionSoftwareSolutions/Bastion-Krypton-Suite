Bastion Krypton Suite

Getting Started — a friendly first-run guide for beginners

Welcome. The Bastion Krypton Suite is a large set of modern Windows Forms controls — buttons, ribbons, docking, navigators, themes and hundreds of extended widgets — so your desktop apps can look polished with very little effort. This page tells you the quickest ways to see it and to use it.

1. Just want to see what it can do?

You don't need Visual Studio for this. Open the Start Menu group Bastion Krypton Suite and launch either of these ready-to-run apps:

Krypton Explorer (Browse Demos)

A launcher for the sample gallery — open it and click through live examples of every major control.

Theme Browser

Flip through all the built-in themes (Office 365, Sparkle, Professional and more) and watch the controls restyle instantly.

Extended Toolkit Examples

A showcase of the extended widgets: toast notifications, fancy grids, progress bars, colour pickers and more.

One-time requirement: the ready-to-run demos need the free .NET 8 Desktop Runtime from Microsoft. If a demo doesn't start, install it from https://dotnet.microsoft.com/download/dotnet/8.0 (choose “.NET Desktop Runtime 8.x — Windows x64”), then try again.

2. Using it in your own project

The easiest, recommended path is NuGet. In Visual Studio, right-click your project → Manage NuGet Packages → search for Bastion.Krypton.Toolkit and install it. It is a drop-in replacement for the upstream Krypton Toolkit — the namespaces are unchanged, so existing Krypton code just recompiles.

Here is a complete, six-line WinForms window using a Krypton button:

using Krypton.Toolkit;

var form   = new KryptonForm { Text = "Hello Bastion" };
var button = new KryptonButton { Text = "Click me", Location = new Point(20, 20) };
button.Click += (s, e) => KryptonMessageBox.Show("It works!");
form.Controls.Add(button);
Application.Run(form);

Prefer to reference the DLLs directly (offline / no NuGet)? They are installed under lib\<framework>\ in this folder — pick the folder matching your target framework (for example lib\net8.0-windows\) and add a reference to Bastion.Krypton.Toolkit.dll. See DesignerIntegration.md for adding the controls to the Visual Studio Toolbox.

3. Where everything lives

  • lib\<framework>\ — the class libraries (all 11 target frameworks), with XML docs and PDB symbols.
  • lib-extended\<framework>\ — the Extended Toolkit assemblies.
  • Samples\ — the full sample source code (C# and VB.NET) to open and study.
  • Samples-Compiled\ — the ready-to-run apps (Krypton Explorer, Theme Browser, Extended Examples).
  • Documentation\ — the CHM help, the PDF manual, and the HTML help site (open Documentation\html\index.html).

The Start Menu also has a “Bastion Krypton Suite Folder” shortcut that opens this install folder in File Explorer.
